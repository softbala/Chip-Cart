PcMate .NET API

1. Install .NET 7 SDK.
2. Update server/appsettings.json with keys.
3. dotnet restore
4. dotnet run
5. The API listens on port 5000 by default when running with Kestrel/proxy.

Notes:
- Paytm checksum helper is a simplified implementation for example purposes. For production, use Paytm SDK or validated checksum.
